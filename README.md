# 🌸 StudyBloom – Cute Flashcard Generator

A beautiful pastel-themed flashcard generator made using only HTML, CSS and JavaScript.

No APIs.
No frameworks.
100% Frontend.

---

## ✨ Features

🌸 Cute Pastel UI

📚 Flip Flashcards

🌙 Dark Mode

🔍 Search Cards

💾 Local Storage

📤 Export TXT

📥 Import Notes

📊 Progress Tracker

🎉 Confetti Celebration

📱 Fully Responsive

---

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript
